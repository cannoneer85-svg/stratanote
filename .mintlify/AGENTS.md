# Mintlify Documentation Agent Rules

## Scope Guardrails
- **CRITICAL**: Do NOT modify, write, or delete any files in the `docs/api-reference/` directory. These files are programmatically auto-generated from source code using TypeDoc.
- You are allowed to suggest edits, draft updates, and create new documentation files/folders inside the `docs/user-guides/` directory and in the root of the `docs/` directory (excluding the forbidden `docs/api-reference/` directory).

## Changelog Policy
- Do NOT run automations to generate or update the changelog (e.g., Draft changelog). The changelog is managed externally via repository release workflows.
