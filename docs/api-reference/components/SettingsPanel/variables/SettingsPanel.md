System configuration settings and administrative dashboard panel component.
Organizes management tasks across multiple settings tabs:
- General: Localization setup.
- Import & Upload: ZIP backups extraction and merge modes.
- Users: Approvals, registrations, and access control role selection.
- Media Library: File grid explorer, size analysis, and orphaned file filters.
- Synchronization: API token generation and vector search indices rebuilding.
- Trash Bin: Recovering notes and historical version logs.
- System: Checking update versions, showing system environment diagnostics, and link checkers.
