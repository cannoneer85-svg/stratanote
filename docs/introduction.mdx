**StrataNote** combines the best features of classic note-taking applications (such as Obsidian) with powerful real-time collaboration and web administration capabilities.

All data is stored in the open Markdown format, preventing vendor lock-in and guaranteeing the privacy of your information.

## Core Features

<CardGroup cols={2}>
  <Card title="Collaboration" icon="users" color="#8b5cf6">
    Real-time intelligent file locking prevents editing conflicts. Review mode (Suggest Mode) allows proposing changes without locking the document.
  </Card>
  <Card title="Connection Graph" icon="network-wired" color="#8b5cf6">
    Interactive 2D visualization based on a Force-directed graph links your notes into a unified semantic web of knowledge.
  </Card>
  <Card title="AI Search & Connections" icon="brain" color="#8b5cf6">
    Fast full-text search (SQLite FTS5) is enhanced by semantic search based on local neural vector embeddings.
  </Card>
  <Card title="Version Control" icon="clock-rotate-left" color="#8b5cf6">
    Second-by-second history of changes for every file with visual comparison (Diff) and instant rollback to any point in the past.
  </Card>
</CardGroup>
